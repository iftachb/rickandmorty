The creation process includes the following:
1. create empty project
2. create model
3. create application class and initialize the koin and the model inside
4. create the main fragment and init the fragment form the main activity
5. create the communication layer. Connect it by the model
6. create share view model and listen the parameters from communication to fill the details
7. create the main screen UI includes RecyclerView
8. create adapter, holder and item xml. Fill the item from the adapter holder class
9. form the holder move to the CharacterDetailsScreenFragment fragment (using animation) - pass the position
10. implement the CharacterDetailsScreenFragment and character_details_screen.xml
11 fill the character_details_screen's field using the shared view model and position
